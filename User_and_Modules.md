User and Module Identification

Users

Student

- Register and Login
- Post academic doubts
- Search doubts
- View answers
- Access learning resources

Faculty

- View doubts
- Answer doubts
- Manage doubt categories

Administrator

- Manage users
- Monitor system activities
- Manage subjects and categories

---

Modules

1. User Management Module

Handles registration, login, and profile management.

2. Doubt Submission Module

Allows students to submit academic doubts.

3. Doubt Resolution Module

Allows faculty to answer doubts.

4. Search Module

Allows users to search solved doubts.

5. Resource Recommendation Module

Provides relevant study materials.

6. Administration Module

Manages users, categories, and system settings.
